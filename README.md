# Como criar um repositório no Github
É necessário ter intalado o GIT no computador.
Vamos usar também o **GitHub Desktop** para facilitar.

## Passo a passo
Supondo que você já temum projeto criado,  faça o seguinte
- Abrir o CMD dentro da pasta e digitar o comando ```git init```
- Ir no Github Desktop e escolher a opção ```Add existing repository``` e selecionar a pasta do projeto
- POrt fim, fazer o commit e publicar. **Atenção:** lembrar de deixar semore públivo
